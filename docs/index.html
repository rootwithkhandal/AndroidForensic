Acquisition Modules & Artifact Engines

Every module executes non-destructively over ADB, automatically creating SQLite tables, CSV spreadsheets, JSON records, and visual analytical maps.

🗺️
Geolocation & Journey Map
Extracts GPS coordinates (`dumpsys location`), EXIF photo coordinates, and geocoded call/calendar strings into industry-standard taxonomy (`Visited`, `POI`, `Media`, `Other`) with interactive Leaflet HTML maps and Google Earth KML files.
--journey journey_map.html
🌐
Multi-Browser History & URLs
Extracts search queries (`q=...`) and visited URLs across Google Chrome, Samsung Internet, Mozilla Firefox, Microsoft Edge, Brave, and Opera via Content Providers and `dumpsys activity recents`.
--browser-history browser_visited_urls.db
🍪
Web & System Cookies
Harvests HTTP cookies from browser profile databases (`Cookies`, `cookies.sqlite`) and Android System WebViews (`app_webview/Default/Cookies`), exporting to SQLite and standard Netscape HTTP Cookie File format (`cookies_netscape.txt`).
--cookies cookies_netscape.txt
⚙️
System & Device Events
Reconstructs exact minute-by-minute user timelines including Screen Unlock (`KEYGUARD_HIDDEN`), Screen Lock (`KEYGUARD_SHOWN`), app foreground transitions (`usagestats`), and system reboots (`bootstat`).
--device-events device_events_timeline.db
🔌
Connected Peripherals
Catalogs physical and wireless hardware including USB accessories/cables, Bluetooth paired/bonded gear (`MAC`, `Name`), Wi-Fi P2P peers, Hotspot clients, companion wearables (Galaxy Watch), and smart cast TVs.
--connected-devices connected_devices.db
🌐
Network Diagnostics
Dumps network interfaces (`ip addr show`), hardware MAC addresses (`wlan0`), live TCP/UDP sockets (`netstat/ss`), local ARP neighbor tables, routing rules, and per-app network usage statistics.
--network network_interfaces.db
📞
Call Logs Acquisition
Queries `content://call_log/calls` and `dumpsys calllog` to extract incoming, outgoing, missed, and rejected calls with exact timestamps, contact names, and call durations.
--call-logs call_logs.db
💬
Messages & Instant Messaging
Acquires standard SMS messages (`content://sms`) and parses WhatsApp Web Companion (`whatsapp_companion_raw.json`), Telegram, and Signal communication caches.
--messages messages.db
👥
Contacts & Address Book
Queries `content://com.android.contacts/data/phones` and merges contact names, phone numbers, email addresses, and WhatsApp JID profiles into comprehensive registries.
--contacts contacts.db
🔔
Notification History
Captures unredacted live and historical system notification queues (`dumpsys notification --noredact`), recovering notification text sent by WhatsApp, Gmail, banks, and social media.
--notifications notifications.db
📅
Calendar & Schedule Export
Extracts calendar events (`content://com.android.calendar/events`), titles, descriptions, and location strings into SQLite, CSV, JSON, and standard iCalendar (`calendar_export.ics`) files.
--calendar calendar_export.ics
🔒
Cryptographic Integrity
Computes SHA-256 and MD5 cryptographic hashes for every single acquired file, saving verifiable manifests (`hash_manifest.json/.csv`) and HTML court-ready summaries.
Automatic hash_manifest.json
